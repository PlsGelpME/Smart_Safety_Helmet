Pre-requisite: Turn ON mobile Hotspot or WiFi and connect Laptop to that hotspot or WiFi.
Step 1: Start Broker
	->Open CMD as Admin
	->Stop mosquitto.exe 
		net stop mosquitto
	->Change directory to mosquito folder
		cd C:\Program Files\mosquitto
	->Run mosquito with configuration file
		mosquito -c path\of\mosquitto.conf -v

Step 2: Start Server Side script
	-> Run main_server.py file from Git Repo/Smart_Safety_Helmet/Central-system-code

Step 3:Power ESP32 and ARD UNO
	-> Open Thonny python editor
